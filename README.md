## Описание

Сервис поиска фильмов с лендингом портфолио автора.

Общий функционал:

- авторизация и регистрация пользователя;
- редактирование данных пользователя;
- поиск фильмов по названию и длительности;
- сохранение фильмов в личной коллекции и удаление фильмов

Проект помог развить навыки backend-разработки, работы с API, развёртывания приложения на Node.js и Express, а также работе с MongoDB & Mongoose.

Репозиторий frontend проекта https://github.com/Annnek/movies-explorer-frontend.

## Функциональность

- Создание схем и моделей пользователей и карточек с фильмами
- Авторизацией защищены все маршруты, кроме страницы регистрации и логина.
- Аутентификация с JWT токеном
- Производится валидация поступающих данных:
  - до передачи информации контроллерам с помощью celebrate
  - на уровне схем с помощью validator и встроенных методов mongoose
- Мидлвэр для централизованной обработки ошибок.
- Реализовано логирование запросов и ошибок
- Реализован ограничитель количества запросов (защита от DDoS атак)
- Работа с CORS

## Директории

`/controllers` — файлы контроллеров пользователя и фильма

`/errors` — файлами кастомных ошибок

`/middlewares` — с мидлварами

- аутентификация;
- центральный обработчик ошибок;
- логгер;
- валидация;
- ограничитель количества запросов;

`/models` — файлы описания схем пользователя и фильма

`/routes` — файлы роутера

`/utils` — вспомогательные файлы

---

## Запуск проекта

`npm i` — установка зависимостей

`mongod` — запускает mongodDB;

`npm run start` — запускает сервер;

`npm run dev` — запускает сервер с hot-reload.
